# od-universal-front

## Development
```shell script
npm run start
```

## Build
```shell script
# install docker and make
# docker - https://docs.docker.com/install/
# make - https://www.gnu.org/software/make/

# build and run container
make build-n-run
```

## Test
```
npm run test
```
